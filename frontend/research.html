The quantitative stack used to reconstruct human decisions, constrain uncertainty, and project the systematic counterfactual against which the live operator is audited.

01

3-Layer Architecture

Project BWC is organized around three audit layers. Each layer answers a separate question: what happened, what risk remained, and what the systematic engine would have done instead.

01 Forensic Attribution Brinson-Fachler plus actual-versus-model execution reconstruction
↓
02 Probabilistic Measurement Cornish-Fisher VaR and GBM plus jump diffusion to a freeze date
↓
03 Regime-Aware Signaling Sparse inverse covariance, macro regime fusion, and HRP allocation

The underlying implementation remains modular, but the public frame is deliberately narrower: isolate behavior, measure risk, and compare discretionary action against a disciplined model path.

02

Regime-Aware Signaling

The signaling layer redistributes exposures as macro conditions change. Five regimes govern how the systematic benchmark should rotate before the human operator overrides it:

EXPANSION

Risk-on. Momentum and carry signals dominate. Full equity exposure with growth tilt.

RECOVERY

Early cycle. Value signals gain weight. Gradual re-risking from defensive positioning.

SLOWDOWN

Late cycle. Volatility signals increase. Defensive rotation, quality factor emphasis.

CONTRACTION

Risk-off. Maximum hedging. Treasury duration extended, equity beta minimized.

CRISIS

Tail event. All signals suppressed. Portfolio collapses to minimum-variance allocation with cash buffer.

Weight redistribution follows a softmax transformation over regime-conditional signal performance, ensuring smooth transitions rather than hard switches between allocation regimes.

03

Hierarchical Risk Parity

The allocation pipeline uses a three-stage process to construct the covariance-aware portfolio:

A
GraphicalLassoCV

Sparse precision matrix estimation via L1-penalized maximum likelihood. Cross-validated to select optimal regularization, producing a denoised covariance structure.

B
Hierarchical Risk Parity

López de Prado's HRP algorithm clusters assets by correlation distance, then allocates risk inversely proportional to cluster variance via recursive bisection.

C
Minimum Spanning Tree

The MST extracted from the correlation matrix provides the dendrogram for hierarchical clustering, revealing the true dependency structure of the asset universe.

04

Monte Carlo Forward Simulation

Forward portfolio paths are generated using Geometric Brownian Motion augmented with a compound Poisson jump-diffusion process. The stochastic differential equation governing the asset price is:

$$dS_t = \mu S_t \, dt + \sigma S_t \, dW_t + S_t \, dJ_t$$

where $W_t$ is a standard Wiener process and $J_t$ is a compound Poisson process with intensity $\lambda$ and log-normal jump sizes $\ln(1+J) \sim \mathcal{N}(\mu_J, \sigma_J^2)$.

10,000 paths are simulated over the 29-day forward horizon. The Monte Carlo hurdle represents the fraction of paths exceeding the risk-free rate, providing a probabilistic confidence measure for the portfolio's forward outlook.

05

Forensic Attribution

Returns are decomposed using the Brinson-Fachler model, separating total active return into allocation, selection, and interaction effects:

$$R_{\text{active}} = \underbrace{\sum_i (w_i^p - w_i^b) R_i^b}_{\text{Allocation}} + \underbrace{\sum_i w_i^b (R_i^p - R_i^b)}_{\text{Selection}} + \underbrace{\sum_i (w_i^p - w_i^b)(R_i^p - R_i^b)}_{\text{Interaction}}$$

This decomposition is used as an audit instrument, not a victory lap. It reveals whether performance came from the intended exposures, from incidental factor drift, or from discretionary deviation away from the model.

06

Modern Metrics

Beyond classical Sharpe, the system computes three modern statistical tests to guard against overfitting and data-mining bias:

Probabilistic Sharpe Ratio (PSR)

Tests whether the observed Sharpe ratio is statistically distinguishable from a benchmark, accounting for skewness and kurtosis of the return distribution.

Deflated Sharpe Ratio (DSR)

Adjusts the Sharpe ratio for the number of trials conducted, penalizing strategies that emerge from extensive backtesting over many parameter combinations.

Probability of Backtest Overfitting (PBO)

Uses combinatorial symmetric cross-validation to estimate the probability that the optimal in-sample strategy underperforms out-of-sample.

Key Equations

Kelly Criterion — optimal fractional sizing under uncertainty:

$$f^* = \frac{p}{a} - \frac{q}{b} = \frac{pb - qa}{ab}$$

where $p$ is win probability, $q = 1-p$, $b$ is win/loss ratio, and $a$ is the loss amount.

Cornish-Fisher Expansion — VaR adjusted for non-normality:

$$z_{CF} = z_\alpha + \frac{1}{6}(z_\alpha^2 - 1)S + \frac{1}{24}(z_\alpha^3 - 3z_\alpha)(K - 3) - \frac{1}{36}(2z_\alpha^3 - 5z_\alpha)S^2$$

where $S$ is skewness, $K$ is kurtosis, and $z_\alpha$ is the standard normal quantile.
